# Library Management System

A Java-based Library Management System developed using Core Java, JDBC, and MySQL.

## Features
- Add and view books
- Add and manage students
- Issue and return books
- Track book availability

## Technologies Used
- Java
- JDBC
- MySQL

## Project Structure
- DAO pattern
- Model classes
- JDBC connectivity

## How to Run
1. Import project into Eclipse / VS Code
2. Add MySQL Connector JAR to build path
3. Create database using `library.sql`
4. Update DB credentials in `DBConnect.java`
5. Run `LibraryMain.java`
